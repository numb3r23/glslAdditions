glslAdditions
=============

Various useful stuff for working with GLSL shaders.

### GLSL-filter for doxygen ###

A filter for [doxygen](http://www.doxygen.org/) to process GLSL shader, usage explained [here](http://www.numb3r23.net/2012/02/15/doxygen-and-glsl-shader/). An example can be seen [here](http://www.grasmo.de/download/glslfilter/glslfilter_example.html)


### LaTeX Syntax-highlight with listings ###

If you want some colored source code in your LaTeX document feel free to use the syntax-definitions.
